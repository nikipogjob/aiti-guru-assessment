
# Aiti Guru - Test Assignment


## Описание

Одностраничное приложение для управления списком товаров с авторизацией, поиском, сортировкой и возможностью добавления новых позиций. Реализовано с упором на production-подход и масштабируемую архитектуру. Проект выполнен в рамках тестового задания и демонстрирует подход к архитектуре, работе с серверными данными и пользовательским состоянием.

## Демонстрация

https://nikipogjob.github.io/aiti-guru-assessment/


## Функциональность

- Валидация полей при входе
- Возможность запомнить текущую сессию
- Загрузка данных с API
- Поиск товаров
- Сортировка и сброс сортировки
- Сохранение состояния сортировки
- Добавление нового товара
- Toast-уведомления

## Технологический стек

- React
- TypeScript
- Vite
- TanStack Query
- Zustand
- React Hook Form + Zod
- SCSS Modules


## Архитектура

Проект построен по принципам Feature-Sliced Design:

- shared - переиспользуемые компоненты и утилиты
- features - бизнес-функциональность (auth, products)
- pages - композиция экранов

## Установка и запуск

```bash
git clone https://github.com/nikipogjob/aiti-guru-assessment.git
cd aiti-guru-assessment
npm install
npm run dev
```

Открыть http://localhost:5173
